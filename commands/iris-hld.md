Generate or regenerate iris architecture diagrams
using the iris-hld-gen agent at .claude/agents/iris-hld-gen.md.

Arguments: $ARGUMENTS (context | container | security | flow | harness | integration | all)

1. If no argument provided, default to "all".
2. Read the iris setup guide and existing diagrams.
3. Delegate to the iris-hld-gen agent with the requested diagram type.
4. Report which diagrams were generated or updated.
